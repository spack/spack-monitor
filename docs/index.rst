========
Spackmon
========

.. image:: https://img.shields.io/badge/spack-slack-blue
    :alt: Slack
    :target: https://spack.slack.com/

.. image:: https://img.shields.io/github/stars/spack/spack-monitor?style=social
    :alt: GitHub stars
    :target: https://github.com/spack/spack-monitor/stargazers


Spack Monitor, or "spackmon" is a monitoring service and database for Spack.
It will allow you to capture complete output, error, specs, and other metadata
associated with spack builds, either to explore or further query.


.. _main-support:

-------
Support
-------

* In case of **questions**, please post on `stack overflow <https://stackoverflow.com/questions/tagged/spack>`_.
* To **discuss** with other spack users, you can use the `mailing list <https://groups.google.com/g/spack?pli=1>`_. **Please do not post questions there. Use stack overflow for questions.**
* For **bugs and feature requests**, please use the `issue tracker <https://github.com/spack/spack-monitor/issues>`_.
* For **contributions**, visit Spack Monitor on `Github <https://github.com/spack/spack-monitor>`_.
